Multiprogramming with Fixed Tasks (MFT)

MFT is a memory management technique where the memory is divided into fixed-size partitions, and each job is assigned to a partition. The size of each partition does not change during execution.

  • Advantages
    • Simple to implement â€” a process is placed into a partition without worrying about fragmentation at allocation time.
  • Disadvantages
    • Internal Fragmentation: Small processes still occupy an entire partition, wasting memory.
    • External Fragmentation: Unused spaces across partitions cannot be combined to allocate larger processes.
